# 台积电 — [相关动态](../../%E5%85%AC%E5%8F%B8/%E4%B8%AD%E7%A7%91%E6%9B%99%E5%85%89/%E7%9B%B8%E5%85%B3%E5%8A%A8%E6%80%81)

## 核心问题
- 台积电主要先进制程客户（如苹果、[英伟达](../../%E5%85%AC%E5%8F%B8/%E8%8B%B1%E4%BC%9F%E8%BE%BE/%E5%85%AC%E5%8F%B8%E5%8A%A8%E6%80%81)、[AMD](../../%E5%85%AC%E5%8F%B8/AMD/%E5%85%AC%E5%8F%B8%E5%8A%A8%E6%80%81)）的订单流向变化及份额稳定性，以及竞争对手（三星、英特尔）在2纳米及以下节点的实质突破与客户获取进展。
- 台积电在2纳米（N2）及更先进制程（如A14、A10）的研发与量产时间表是否符合预期，以及在先进封装（如[封测](../../%E8%A1%8C%E4%B8%9A/%E5%B0%81%E6%B5%8B/%E5%B0%81%E6%B5%8B)）领域的产能扩张进度与技术壁垒维持情况。
- 公司季度毛利率的走势及驱动因素，特别是在高资本支出和折旧压力下，先进制程的盈利能力与整体资本回报率（ROIC）的变化。
- 全球AI/HPC（高性能计算）需求增长对台积电营收贡献的持续性，以及公司在美、日、欧等地新晶圆厂的建设进度、产能爬坡规划和地缘政治风险下的供应链韧性。
