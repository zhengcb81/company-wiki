# 3位留美博士创办的这家半导体材料企业，上市了—新闻—科学网

| |  | | --- | | 作者：赵广立 来源：中国科学报 发布时间：2024/8/16 21:26:30  选择字号：小 中  大 | | |  | | --- | |  | |  | | 3位留美博士创办的这家半导体材料企业，上市了 | |  |     8月16日，由三位留美博士创办的半导体材料科技企业——苏州珂玛材料科技股份有限公司（简称珂玛科技）正式在创业板挂牌上市，发行价8元/股，首日涨幅高达368.25%，首日报收37.46元/股，据此对应最新市值超163亿元。  敲钟上市。珂玛科技 供图    ? 招股书显示，珂玛科技的创始人刘先兵出生于1969年，2001年从美国康州大学毕业，并取得机械工程博士学位。2009年，刘先兵归国创业，珂玛科技的前身“苏州珂玛材料技术有限公司”成立，系当时国内首家拥有自主知识产权、从高精密陶瓷部件设计制造到清洗维护的供应商。  珂玛科技另外两位灵魂人物——副总经理、研发部负责人施建中和首席科学家及战略项目总监王冠，也都有留美博士背景。施建中毕业于加利福尼亚大学圣地亚哥分校材料科学专业，王冠毕业于纽约州立大学石溪分校材料学专业。两人先后被刘先兵在创业之初抛来的“橄榄枝”拉“入伙”。  三人携手十多年来，共同将珂玛科技打造成了国内先进陶瓷材料的“独角兽”。据了解，该公司研发的陶瓷材料是一种用于制作高性能陶瓷产品的人工合成的无机化合物，在医药、汽车、半导体、锂电池等领域均有应用。而随着人工智能的爆火，泛半导体设备零部件对陶瓷材料的需求放大，但国内仅有少数企业能够提供符合要求的产品，珂玛科技就在其中。因此，珂玛科技同时被贴上了“半导体材料”“国产替代”等概念。  据“天眼查”信息显示，自成立以来，珂玛科技先后完成六次增资及四次股份转让。其中在2021年10月~11月，珂玛科技一举引入18家外部机构，聚集了一串长长的投资方名单，包括北京集成电路装备产业基金、英诺天使基金、华业天成、中金资本、中小企业发展基金、浦东科创集团、苏高新金控、君桐资本、盛芯产投、沃衍资本、钟鼎资本等相关产业资本和市场化投资机构。这也是珂玛科技首次公开募股（IPO）前的唯一一轮大规模融资。  值得一提的是，珂玛科技于2022年6月递交招股书，直到2024年4月才获准提交注册。不过，珂玛科技提交注册后仅8天就拿到批文，刷新了2024年以来A股IPO最快注册审核速度。
